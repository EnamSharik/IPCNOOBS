Conceptos para implementar CSS en tus HTML

Es importante saber que CSS no es un lenguaje de programación, sino que es un lenguaje de diseño gráfico. Este permite proporcionar estilos a una página WEB, ya que permite mostrar una forma más atractiva y ordenada dentro de la página ya sea que fuera realizado con HTML, XHTML, XML, SVG, XUL, RSS, etc. Este lenguaje es muy utilizado en la creación de las páginas web junto a HTML y JavaScript. Este proporciona características principales, entre ellas, se destacan las que tienen que ver con sombras (a texto y cajas), bordes (colores, imagen y redondeado), colores y opacidad, trabajo sobre texto, fuentes, pseudoelementos, rotación, propiedades de interfaz de usuario, transformación, transición y animación, entre otras.
